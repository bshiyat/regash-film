# Regash - Filmmaker Portfolio

Modern portfolio website for filmmaker Alaa Aliabdallah (Regash).

## Structure

- `client/` - Main application (React + Vite + TypeScript)
- All source code is in the client folder

## Vercel Deployment

**Settings:**
- Root Directory: `client`
- Build Command: `npm run build`
- Output Directory: `dist`

## Local Development

```bash
cd client
npm install
npm run dev
```
